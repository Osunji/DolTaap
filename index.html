기본 규칙 :
Basic Rules :
  • 쌓은 돌의 일부가 골라인에 닿은채로 10초를 버티면 성공
  • 돌을 많이 쌓을수록 높은 점수를 획득
  • 최대 콤보를 달성할수록 보너스 점수를 추가로 획득
  • 돌이 화면밖으로 나갈 때 점수가 차감되며 콤보도 초기화 됨
  • The game ends if any part of the stacked stones touched the goal line and remains there for 10 seconds.
  • Higher scores are awareded for stacking more stones.
  • Bonus score are given for achieving the max combo.
  • Scores are deducted and the combo is reset if a stone falls off the screen.
